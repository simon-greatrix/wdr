Werewolf Dice Roller

Do you never have enough D10s? Can you never remember what you are suppose to roll for something? Tired of all the adding and counting involved? The Werewolf Dice Roller is here to help!

Launch The App

Installing

The Werewolf Dice Roller (WDR) is just a web-page. You don't need to install anything. Unless you want to use it where there is no wifi. In that case good news because you can install it, because it is a "Progressive Web App".

On an iPhone, open WDR in Safari, hit the "Share" button and scroll down to find "Add to home screen".

On an android phone, open WDR in Chrome, and there should be an obvious install button.

On a PC, Mac or tablet, there should be a button somewhere. In Chrome it is in the address bar.

Using

At the top of the screen are five icons: Dice, Person, Heart, User Settings, and History. Each of these activates a different function.

Basic Rolling (The Dice icon)

Set the number of dice you want to roll, set the difficulty, see the score.

The arrows raise and lower the number of dice and the difficulty. If you want to change the value a lot, hit the number and a nice pop-up will allow you to type the value you want.

Hit the big '?' to roll, or whatever value is display in the bottom half of the screen. If you get a skull, it means you botched the roll.

If you want to see what the numbers on the dice actually were, hit the History icon and see your last twenty rolls.

Skill Check Rolling (The Person icon)

This screen is only useful after you have entered your character's attributes and abilities. To do that, hit the User Settings icon and enter them.

Set the number of dice you want to roll by saying which attribute and ability your character is using, and what form they are in. The numbers in parentheses show what each is contributing.

Set the difficulty with the arrows, or tapping on the number.

Tap the score to re-roll.

Favourite Rolls (The Heart icon)

Some rolls come up over and over again. On this page you can save your favourites.

Scroll up and down the list, find the one you want, and click it. You'll be taken to the "Skill Check Rolling" with the relevant ability, attribute and difficulty set up.

Hitting the "Plus" sign at the top allows you to add new favourites to the list.

You can re-order the list by dragging on the handle icon on the right.

If you get tired of one of the favourites, you can drag it all the way down and drop it into the bin.

Character Details (The User Settings icon)

Select "Attribute", "Ability", then the sub-category. Tap the various dots to change the values for your character.

History (The History icon)

This is list of your recent dice rolls. It is not interactive.
